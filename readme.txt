== Installation ==

To install this plugin:

1. Install the plugin through the WordPress admin interface, or upload the plugin folder to /wp-content/plugins/ using ftp.
2. Activate the plugin through the 'Plugins' screen in WordPress. On a Multisite you can either network activate it or let users activate it individually. 
3. Go to WordPress Admin > Reunion Website > Settings

== Frequently Asked Questions ==

= How do I use the plugin? =
1. Install the plugin through the WordPress admin interface.
2. Go to the AlumniOnline Reunion Website > Settings page
3. Set the main menu value
4. Review or change the default message board and classmate search pages.
5. Review or change additional settings as desired.
6. Enable "Anyone can register" on the General Settings page.
